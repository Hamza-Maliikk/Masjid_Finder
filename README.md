## 🌟 About the Project

**Masjid Finder** is a modern Islamic web application designed specifically for the Muslims of **Karachi**. The core purpose of this platform is to help any Muslim — whether a regular worshipper or a traveller — find the nearest mosque using their GPS location with ease.

This is not just a mosque finder — it is an **all-in-one Islamic hub** where:

- 📍 **Nearby mosques** can be found using GPS
- 🤲 **Daily duas** are available in one place
- 📖 **Surah Al-Kahf** can be recited easily every Friday
- 🛒 **Islamic products** can be purchased online

> *"Putting technology in the service of Deen — that is the vision of this project."*

## ⚙️ How It Works

### 📍 Finding a Mosque

1. Open the website: [masjid-finder-six.vercel.app](https://masjid-finder-six.vercel.app/)
2. Click the **"Detect"** button on the home page
3. A browser location popup will appear — click **"Allow"**
4. Your GPS coordinates will be detected automatically
5. Nearby mosques will be listed along with their distances

### 🤲 Reading a Dua

1. Click **"Dua"** in the top navigation
2. The list of duas will load
3. Type the dua you need in the search bar
4. Read your desired supplication

### 📖 Reciting Surah Al-Kahf

1. Click **"Surah"** in the top navigation
2. The full Surah Al-Kahf will load
3. Recite verse by verse

### 🛒 Shopping from the Store

1. Click **"Store"** in the top navigation
2. Select a product category (Prayer Mats, Quran, etc.)
3. Add products to your cart
4. Review your cart and proceed to checkout

## 🚀 Installation

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/Hamza-Maliikk/Masjid_Finder.git

# Navigate into the project folder
cd Masjid_Finder

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open your browser at: `http://localhost:3000`

### Build for Production

```bash
npm run build
npm start
```
